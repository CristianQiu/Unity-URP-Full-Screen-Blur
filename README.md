# About

* Unity package that includes a render feature to do a full screen blur.
* Integrated with URP volume system.
* Unity 6000.3.0f1 or above is required.

# Usage

* Add the full screen blur renderer feature to you URP Renderer.
* Enable post-processing both in your camera and URP Renderer.
* Add a volume to your scene and add override: "Custom->Full Screen Blur".
* Tweak the parameters to your needs to blur the screen.
